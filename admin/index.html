使用说明

1. 发布新文章流程：

在"发布文章"页面填写信息 → 点击"生成JSON" → 复制生成的JSON数据

2. 更新数据文件：

将复制的JSON数据粘贴到 data/articles.json 文件中，替换原有内容

3. 部署到 Vercel：

修改后的 articles.json 提交到 Git 仓库，Vercel 会自动重新部署

4. 文件路径：

项目根目录 / data/articles.json
